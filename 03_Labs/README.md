# Cybersecurity Labs

This directory contains hands-on labs designed to understand how network protocols behave and how they can be analyzed from a security perspective.

The labs progress from fundamental network protocols to secure protocols and attack scenarios.

## Structure

The labs are organized by protocol categories:

```
01-basic-protocols
02-transport-layer-protocols
03-infrastructure-protocols
04-tls
05-file-transfer-protocols
06-email-protocols
07-remote-access-protocols
08-attack-labs
```

Each category contains several labs focusing on a specific protocol or concept.

## Lab Format

Every lab follows a consistent structure:

- Objective  
- Lab Environment  
- Network Configuration  
- Tools Used  
- Procedure  
- Observation  
- Security Relevance  
- Conclusion  

Screenshots and evidence for labs are stored in the `screenshots` folder inside each category.
