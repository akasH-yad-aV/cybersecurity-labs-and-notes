# Cybersecurity Labs & Notes Repository

## Overview
This repository contains a structured collection of cybersecurity and networking work, combining both theoretical understanding and hands-on practical labs.

It includes detailed notes and real packet-level analysis to understand how systems and protocols behave in actual environments.

---

## Key Areas Covered

### 🔹 Networking & Protocol Analysis
- TCP/IP, UDP, DNS, DHCP, ARP, ICMP  
- Packet-level inspection using Wireshark  
- TCP handshake, retransmission, flow control  
- DNS query-response and DHCP negotiation  

### 🔹 Operating Systems
- Linux fundamentals (commands, permissions, system basics)  
- Windows system fundamentals and security concepts  

### 🔹 Cybersecurity Fundamentals
- Core security concepts  
- Basic attack types and defensive practices  
- Blue team concepts and SOC fundamentals  

---

## 🔬 Hands-On Labs & Analysis
- Real network traffic capture and analysis using Wireshark  
- Protocol behavior investigation at packet level  
- Practical observation of communication patterns  
- Structured lab documentation with screenshots  

---

## ⚔️ Security & Attack Labs
- Basic attack understanding and simulation  
- Network-based attack concepts  
- Defensive awareness and analysis

## Objective
The goal of this repository is to move beyond theoretical knowledge and build practical skills in:

- Network traffic analysis

This work is aligned with roles such as **SOC Analyst** and **Network Security Analyst**, focusing on real-world analysis rather than just theory.
